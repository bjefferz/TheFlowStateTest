# The Flow State
Code for Justice.

## Project Overview:
Welcome to TheFlowState, a website created for our non-profit organization, developed during the Road to Hire apprenticeship's Code for Justice Hackathon.

The core purpose of this project is to raise awareness of the health impacts associated with non-potable water and to encourage community action. Our primary goal is serve as an online resource for NC residents who are struggling to access clean, potable water in specific, affected areas.

## Features
- **Awareness & Content:** Five dedicated pages featuring content serving as a resource to inform users of affected areas the serverity of non-potable water.
- **Responsive Design:** The website is optimized to display across Desktop and Mobile devices.
- **Client-Side Form Validation:** Connect and Donate pages utilize JavaScript validation to provide appropriate error feedback. _(Implemented by Tommy-Nam and Brendan)_
- **Home Page:** Serves as the default landing page that explains the "why" of our organization. _(Implemented by Christine)_
- **About Page:** Showcases the organizations mission and the team profile. _(Implemented by Marilyn)_ 
- **Connect Page:** Serves to allow communication between the user and organization. _(Implemented by Brendan)_
- **Donate Page:** Allows users to select or enter custom amounts to donate to the organizations cause. _(Implemented by Tommy-Nam)_
- **Resources Page:** Provides users with helpful, detailed information about the non-potable water crisis in North Carolina. _(Implemented by Jessica)_ 

## Usage Guide (Run Locally)
### Prerequisites:
- Git
- VSCode (Open Live Server extension)

### Setup:
1. **Clone the repository:**
   ```bash
   git clone https://github.com/C18-Group4/TheFlowState.git
2. **Navigate and Select the correct root directory (Important for correct routing)**
   ```bash
   File -> Open Folder -> TheFlowState 
4. **Install VSCode's 'Open Live Server' extension**
5. **Inside VSCode, Select ../TheFlowState/pages/index.html**
6. **Right Click 'index.html' and click 'Open with Live Server'**
